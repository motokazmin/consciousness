# consciousness

Экспериментальный **HRV Awareness Monitor** (Polar H10 / mock, RMSSD).

- **Архитектура и назначение:** [ARCHITECTURE.md](ARCHITECTURE.md)
- **Детали MVP** (веб-UI, BLE, baseline, mock): [hrv_mvp.md](hrv_mvp.md)
- **Веб-аудио (Web Audio):** [ARCHITECTURE.md § Веб-аудио](ARCHITECTURE.md#веб-аудио-где-генерируется-звук)

## Быстрый старт

```bash
# зависимости
pip install -r requirements.txt

# веб-интерфейс (форма сессии, live-графики uPlot, архив, прогресс) — единственный UI записи
python -m hrv_web
# откройте в браузере: http://127.0.0.1:8765/
# Укажите участника, тип активности (из списка или «Новая активность…»), источник (Mock / BLE),
# при необходимости MAC и длительность. Типы хранятся в SQLite (таблица session_types).
# Заметки к сессии — см. раздел «Теги заметок» в hrv_mvp.md (#утро, #глубоко и т.д.)
```

## Теги

В системе два разных понятия — не путайте их:

| | **Тип активности** | **Теги заметок** |
|---|-------------------|------------------|
| Где задаётся | Выпадающий список «Тип активности» при старте | В поле «Заметка» или в модальном окне после сессии |
| В БД | `sessions.tag` (slug) | Внутри `sessions.session_name`, формат `#слово` |
| Зачем | Класс практики: медитация, йога, работа… | Свой контекст для поиска: `#утро`, `#устал`, `#эксперимент` |
| Фильтр | «Тип активности» на вкладках Архив / Прогресс | «Теги заметок» — можно отметить несколько (сессии с **любым** из выбранных) |

**Теги заметок** пишутся через `#` прямо в тексте:

```
Глубокое расслабление после 10-й минуты #утро #спокойствие
#эксперимент дыхание 4-7-8, отвлекался на улицу
```

Правила: буквы (в т.ч. кириллица), цифры, `-`, `_`; до 32 символов; регистр не важен (`#Утро` = `#утро`). Пробел после тега завершает его — `#утро день` даёт только тег `утро`.

Подробнее: [hrv_mvp.md § Тегирование](hrv_mvp.md#тегирование-сессий).

**BLE / BlueZ:** если при подключении видите `Bleak requires BlueZ >= 5.55`, в системе старая BlueZ (часто 5.53). Переустановите зависимости: `pip install 'bleak>=0.22.3,<1'`, либо обновите BlueZ в ОС. Для BLE укажите MAC Polar H10 в веб-форме.
